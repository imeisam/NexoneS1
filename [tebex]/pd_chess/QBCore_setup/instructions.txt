in your server files, open: [qb] > qb-core > shared > items.lua

and add this line to the bottom:
['pd_chess_board'] 				 		 = {['name'] = 'pd_chess_board', 			    		['label'] = 'Chess Board', 					['weight'] = 100, 	['type'] = 'item', 		['image'] = 'chess_board.png', ['unique'] = false, 	['useable'] = true, 	['shouldClose'] = true,   ['combinable'] = nil,   ['description'] = 'Care for a game?'},

Put the chess_board.png located in this folder in: [qb] > qb-inventory > html > images
